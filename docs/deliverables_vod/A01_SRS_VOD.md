## 1. 개요

**2026_TV VOD 서비스**는 두 개의 독립 트랙으로 구성된다.

| 구분 | 트랙1 | 트랙2 |
|------|-------|-------|
| 명칭 | 금주의 무료 VOD | 추천 VOD (당신을 위한 무료 VOD) |
| 선정 방식 | 매주 월요일 배치 (슬롯 기반 큐레이션) | 실시간 NLP 개인화 추천 |
| 광고 | FAST 광고 오버레이 포함 | 광고 없음 |
| 대상 | 전 고객 공통 | 개인별 |
| VOD 수 | 10개 | 10개 |

## 2. VOD 페이지 구성 요구사항 (FR-004)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-004-01 | VOD 페이지는 **광고 배너 / 금주의 무료 VOD / 추천 VOD** 3단 레이아웃으로 구성 | 필수 |
| FR-004-02 | 모든 섹션이 한 화면에 표시, 수직 스크롤 비활성화 | 필수 |
| FR-004-03 | 최초 포커스는 **금주의 무료 VOD 섹션 첫 번째 항목**에 위치 | 필수 |
| FR-004-04 | 리모컨 키(`▲▼` 섹션 이동, `←→` 항목 이동, `ENTER` 선택, `ESC` 뒤로) | 필수 |

### 2.1 광고 배너 (FR-004-AD)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-004-AD-01 | 한 번에 1개 배너만 표시하는 캐러셀 방식 | 필수 |
| FR-004-AD-02 | `←→` 키 또는 버튼으로 이전/다음 배너 이동 | 필수 |
| FR-004-AD-03 | **5초마다 자동으로 다음 배너로 전환** (VOD 재생·모달 열림 시 일시 정지) | 필수 |
| FR-004-AD-04 | 하단 인디케이터(점)로 현재 위치 표시 | 필수 |

### 2.2 금주의 무료 VOD — 트랙1 (FR-004-T1)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-004-T1-01 | 전 고객 공통, 주 1회(매주 월요일) 자동 갱신 | 필수 |
| FR-004-T1-02 | 총 **10개** VOD 보유, 화면에는 **6개씩 슬라이딩 윈도우** 표시 | 필수 |
| FR-004-T1-03 | 포커스가 뷰 밖으로 이동 시 윈도우 자동 슬라이드 | 필수 |
| FR-004-T1-04 | 각 카드에 썸네일 이미지, 제목, 장르, 순위 배지(`#N`) 표시 | 필수 |
| FR-004-T1-05 | VOD 선택 시 **전체화면 플레이어**로 전환, FAST 광고 오버레이 적용 | 필수 |
| FR-004-T1-06 | 현재 위치 표시 (`3 / 10` 형식) | 선택 |

**v2 큐레이션 선정 전략** (`ad-batch` 배치, 매주 월요일 02:00):

| 항목 | 내용 |
|------|------|
| **하드 필터** | `USE_FL=1`, `SVC_APY_FL=1`, `THMBNL_FL=1`, `IS_FREE_YN='N'`, `DISP_RTM >= '00:20:00'` (20분 이상) |
| **슬롯 배분** | 키즈/애니 ≤3개 + 교양/다큐 ≤4개 + 예능/음악 ≤2개 + 기타 ≤1개 |
| **소프트 점수** | 인기작(`IS_HOT_FL`) +15, 고화질(`HD/FHD/UHD`) +10, 키즈 1화 유도 +15, 4060 키워드 +20, 월별 테마 +30 |
| **중복 제거** | `SUPER_ASSET_NM` 기준 시리즈당 최고점 1편 선정 |
| **시즌 보너스** | 12개월 테마 키워드 자동 적용 (`seasonal_themes.py`) |
| **IS_FREE_YN 복원** | 신규 배치 전 이전 주 VOD의 `IS_FREE_YN='N'` 자동 복원 |
| **선정 사유 기록** | `SELECTION_REASON`: `SLOT_KIDS` / `SLOT_DOCU` / `SLOT_ENT` / `SLOT_ETC` |

### 2.3 추천 VOD — 트랙2 (FR-004-T2)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-004-T2-01 | 고객 개인화 추천, 실시간 조회 | 필수 |
| FR-004-T2-02 | 총 **10개** VOD, 화면에는 **6개씩 슬라이딩 윈도우** 표시 | 필수 |
| FR-004-T2-03 | VOD 선택 시 전체화면 플레이어 전환 (광고 없음) | 필수 |
| FR-004-T2-04 | 키즈·애니메이션 장르는 연령과 무관하게 추천 풀에 항상 포함 | 필수 |

## 3. NLP 기반 개인화 추천 엔진 (FR-005)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-005-01 | TF-IDF + KeyBERT로 VOD 메타데이터 텍스트 벡터화 | 필수 |
| FR-005-02 | 벡터화 소스: `TTL + GENRE + DESCRIPTION + HASH_TAG + SMRY` | 필수 |
| FR-005-03 | TF-IDF 모델을 pickle 파일(`tfidf.pkl`)로 영속화 — 재시작 시 자동 로드 | 필수 |
| FR-005-04 | 고객 시청이력(`TB_VOD_LOG`) 기반 유저 프로필 벡터 생성 (시청 시간 가중 평균) | 필수 |
| FR-005-05 | 코사인 유사도 기반 상위 10개 VOD 선정 | 필수 |
| FR-005-06 | 키즈·애니메이션 장르에 별도 `kids_boost_score` 가중치 추가 | 필수 |
| FR-005-07 | 한국어 사전학습 모델: `snunlp/KR-ELECTRA-discriminator` | 필수 |
| FR-005-08 | 추천 사유 텍스트 자동 생성 (score 기반 분류) | 선택 |
| FR-005-09 | 서비스 기동 시 TF-IDF 모델 자동 로드, KeyBERT 사전 워밍업 | 필수 |

### 3.1 Cold Start 처리 정책 (FR-005-CS)

| 상황 | 처리 방법 | 결과 |
|------|---------|------|
| 유저 벡터 있음 | 사전 계산된 프로필 벡터로 코사인 유사도 계산 | 개인화 추천 |
| 유저 벡터 없음 + 시청 이력 있음 | `TB_VOD_LOG` 기반 임시 프로필 생성 | 이력 기반 추천 |
| **시청 이력 없음 (완전 신규)** | `RATE` 평점 내림차순 top 10 반환 | 인기 콘텐츠 추천 |

> Cold Start 응답: `reason="인기 콘텐츠 추천 (시청 이력 없음)"`, `score=1.0` 고정

## 4. FAST 광고 파이프라인 (FR-006)

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-006-01 | 매주 월요일 02:00 KST 자동 배치 실행 | 필수 |
| FR-006-02 | PySceneDetect ContentDetector로 씬 분할 (비디오 프레임만, threshold=30.0) | 필수 |
| FR-006-03 | 각 씬 시작점의 대표 키프레임 PNG를 ffmpeg으로 추출 | 필수 |
| FR-006-04 | **YOLOv8n** (COCO 80클래스, conf≥0.5)으로 키프레임 내 객체 감지 | 필수 |
| FR-006-05 | **CLIP** (`openai/clip-vit-base-patch32`)으로 20개 컨텍스트 태그 중 상위 5개 추출 | 선택 |
| FR-006-06 | YOLO + CLIP 결과 통합 → `vision_tags` (중복 제거, 최대 10개) | 필수 |
| FR-006-07 | 저움직임 구간 탐지 → 최적 삽입 포인트 최대 5개 선정 | 필수 |
| FR-006-08 | 삽입 타임스탬프: 씬 시작점 +1.0초, `insert_reason` 자동 분류 | 필수 |
| FR-006-09 | 생성형 AI API로 이미지 광고 생성 (DALL-E 3, 1024×1024 PNG) | 필수 |
| FR-006-10 | 생성형 AI API로 무음 영상 광고 생성 (RunwayML/Kling, 4초 MP4) | 필수 |
| FR-006-11 | API 키 미설정 시 graceful fallback (PIL 플레이스홀더 / ffmpeg 검은화면) | 필수 |
| FR-006-12 | 원본 VOD 파일 무수정 — 클라이언트 AdOverlay 오버레이 방식으로 광고 노출 | 필수 |
| FR-006-13 | 광고 오버레이 위치: 화면 하단, 4초 표시, 동일 타임스탬프 1회만 노출 | 필수 |
| FR-006-14 | `TB_WEEKLY_FREE_VOD.AD_PIPELINE_STATUS`에 파이프라인 상태 기록 | 필수 |
| FR-006-15 | `SELECTION_REASON` 슬롯별 기록: `SLOT_KIDS`/`SLOT_DOCU`/`SLOT_ENT`/`SLOT_ETC` | 필수 |
| FR-006-16 | 슬롯 미달(10개 미만) 시 `weekly_vod.slot_shortage` 경고 로그 출력 | 필수 |
| FR-006-17 | 배치 실행 전 이전 주 VOD `IS_FREE_YN='N'` 자동 복원 | 필수 |

## 5. 핵심 비즈니스 규칙

> ⚠️ **절대 위반 금지**

1. **키즈·애니메이션은 추천 풀에서 절대 배제 금지.** 오히려 주요 추천 요소로 취급.
2. 키즈·애니메이션은 `kids_boost_score` 가중치로 추천 보장.
3. **원본 VOD 파일을 절대 수정하지 않는다.** 광고는 클라이언트 오버레이로만 삽입.
